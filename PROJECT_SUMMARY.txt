================================================================================
                    SHOESPHERE PROJECT SUMMARY
================================================================================

Project Overview:
-----------------
ShoeSphere is a Next.js e-commerce website for showcasing shoes, powered by 
Contentstack CMS. It's built using Next.js 16 with React 19, TypeScript, and 
Tailwind CSS.

================================================================================
                        CORE FEATURES IMPLEMENTED
================================================================================

1. HOMEPAGE (app/page.tsx)
   - Hero section with video/image background support
   - Featured shoes collection display
   - Adidas-style modern design aesthetic
   - Dynamic content pulled from Contentstack

2. PRODUCT CATALOG
   - All shoes listing page (/shoes)
   - Category browsing (/categories, /category/[slug])
   - Dynamic product detail pages (/shoes/[url])
   - Product grids with images, prices, and details

3. MULTI-CURRENCY SUPPORT
   - Currency selector dropdown (USD, EUR, INR) in navbar
   - Contentstack Personalize SDK integration
   - Real-time price updates based on selected currency
   - Currency preference persistence (localStorage)
   - Variant-based pricing system:
     * USD: cs91db6b7e0d7f71e1 (base variant)
     * EUR: csc4ee31b822d1b0d0
     * INR: csb474334af86d3526

4. NAVIGATION & LAYOUT
   - Responsive navbar with:
     * Logo (from Contentstack)
     * Dropdown navigation menus
     * Currency selector
     * Search icon (UI ready)
     * Cart icon (UI ready)
     * User icon
   - Announcement bar (top banner)
   - Footer with configurable columns

5. PRODUCT DETAILS
   - Full product information display
   - Brand, category, seller details
   - Material and care instructions
   - Size options
   - Customer testimonials/reviews display
   - Testimonial submission form

6. API ROUTES
   - /api/shoes - Fetch all shoes with variant support
   - /api/shoes/[url] - Fetch single shoe by URL
   - /api/category/[slug] - Fetch category data
   - /api/testimonials - POST endpoint to submit customer reviews
     (uses Contentstack Management API)

7. CONTENTSTACK INTEGRATION
   - Content Delivery SDK for fetching content
   - Personalize SDK for currency variants
   - Management API for creating testimonials
   - Content type definitions in Content Types/ folder:
     * shoes.json
     * category.json
     * brand.json
     * seller.json
     * material.json
     * testimonial.json
     * homepage.json
     * site_config.json
